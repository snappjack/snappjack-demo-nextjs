Once connected, AI agents can interact with this app using the following tools:

  • dicely.systemInfo.get - Get system information and current app state
  • dicely.dice.plan - Set which dice to keep or roll (array of 5 actions)
  • dicely.dice.roll - Execute the roll for non-kept dice
  • dicely.dice.reset - Reset all dice to initial state
